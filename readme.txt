Веб сайт написан с использованием технологий: Python, Django, HTML, CSS, Bootstrap 5.
Также в проекте используются библиотеки: requests, bs4.

Как воспользовать:
1) клонировать проект (скачать)
2) создать виртуальное окружение
3) в терминале прописать (в основной папке проекта) pip install -r requirements.txt
4) запустить локально сервер, с помощью команд
	cd my_travel_site
	python manage.py runserver

The website is written using technologies: Python, Django, HTML, CSS, Bootstrap 5.
The project also uses libraries: requests, bs4.

How to use:
1) clone the project (download)
2) create a virtual environment
3) in the terminal, register (in the main project folder) pip install -r requirements.txt
4) start the server locally, using
	cd my_travel_site
	python manage.py runserver
